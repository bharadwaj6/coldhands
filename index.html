    Welcome to the official page of coldhands

    Why coldhands?

    Ever wanted to use the most up to date version of webkit without any additional hassle? Just download coldhands and start using it.

    There is no particular reason for why you should use coldhands except the above mentioned reason. If you are looking for something to use daily or some browser to develop upon, this is probably not for you.
